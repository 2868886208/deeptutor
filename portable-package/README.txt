DeepTutor Portable Setup
=======================

What you need before running:
1. Docker Desktop installed: https://www.docker.com/products/docker-desktop/
2. A DeepSeek API Key: https://platform.deepseek.com

Steps:
1. Double-click setup.cmd
2. Wait for image download (5-15 minutes)
3. Open http://localhost:3782 in browser
4. Settings -> Add DeepSeek API Key
5. Double-click launcher.bat for desktop app mode

To stop: docker compose down (in this folder)
To update: setup.cmd again to pull latest image
